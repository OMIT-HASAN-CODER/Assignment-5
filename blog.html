Question-1: What are the different ways to select an element in the DOM?

We can select an element in the DOM using several JavaScript methods like-

1. document.getElementById("id") – Selects an element by its id.
2. document.getElementsByClassName("class") – Selects elements by their class name.
3. document.getElementsByTagName("tag") – Selects elements by their tag name.
4. document.querySelector("selector") – Selects the first element that matches a CSS selector.
5. document.querySelectorAll("selector") – Selects all elements that match a CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent is-

innerHTM - Returns or sets the HTML content inside an element.
innerText - Returns or sets only the visible text.
textContent - Returns or sets all text inside an element.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where a parent element listens for events on its child elements. Instead of attaching event listeners to multiple child elements, we attach a single event listener to the parent.

Question-4: What is event bubbling in the DOM?

Event bubbling is a type of event propagation where an event starts at the target element and bubbles up to the parent elements. In other words, the event is first captured and handled by the target element, then propagated to its parent elements, and so on until it reaches the root of the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

I can create, add, and remove elements using JavaScript like this-

Create an Element - let newElement = document.createElement("div");

Add an Element - let parentElement = document.getElementById("parent");
parentElement.appendChild(newElement);

Remove an Element - let elementToRemove = document.getElementById("child");
elementToRemove.parentNode.removeChild(elementToRemove);
